# Portfolio Risk Analysis using Value-at-Risk (VaR) Methods

## 📌 Project Overview
This project demonstrates the application of **Actuarial Mathematics** to **Market Risk Management**. I implement Value-at-Risk (VaR) models to quantify potential portfolio losses at specific confidence levels, transitioning theoretical probability concepts into financial engineering practices.

## 🛠️ Technical Implementation
### Data Engineering (SQL)
- Utilized **Advanced SQL (Window Functions, CTEs)** to extract and process high-frequency time-series data from a simulated banking database.
- Calculated **Daily Log Returns** and performed data normalization for statistical modeling.

### Quantitative Modeling
- **Parametric VaR:** Implemented using Variance-Covariance matrices, leveraging **Probability Density Functions (PDF)**.
- **Historical Simulation:** Analyzed non-normal distributions to account for "fat-tail" risks in financial markets.
- **Mathematical Foundation:** This implementation applies core concepts from **Multivariable Calculus** and **Stochastic Processes** (Probability Theory) studied during my Actuarial Science degree.

## 🎯 Key Insights
- **Risk Metrics:** Evaluated 95% and 99% VaR for a diversified asset portfolio.
- **Backtesting:** Validated model accuracy by comparing predicted VaR against actual realized P&L.
